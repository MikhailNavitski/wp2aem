## Prerequisites

1. Java JDK 11
2. AEM cloud-ready SDK 


## AEM Deployment

1. Clone the `wp2aem` project.

        $ git clone https://github.com/headwirecom/wp2aem.git
        $ cd wp2aem

2. Deploy `wp2aem` to AEM.

        $ mvn clean install -PautoInstallPackage -Padobe-public
